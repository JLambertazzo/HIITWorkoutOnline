Run group workouts with a single command.

Try it Yourself Learn More

Install:

Click 'Try it Yourself' above to add the bot to a server

Start a HIIT Workout:

Enter the following command into your discord chat:

!hiit #rounds #high #low [#warmup] [#cooldown]

Using the following arguments

Argument Meaning Example
#rounds Number of Rounds 5
#high Seconds of High-Intensity 45
#low Seconds of Low-Intensity 15
#warmup (optional) Seconds of Warmup 180
#cooldown (optional) Seconds of Cooldown 180

Start a Custom HIIT Workout:

Enter the following command into your discord chat:

!myhiit #rounds #high #low #warmup #cooldown exercises...

Using the following arguments

Argument Meaning Example
#rounds Number of Rounds 5
#high Seconds of High-Intensity 45
#low Seconds of Low-Intensity 15
#warmup Seconds of Warmup 180
#cooldown Seconds of Cooldown 180
exercises... Exercises in the Workout pushups plank
